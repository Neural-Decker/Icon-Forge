# Clockwork Pixie Icon Forge
Generate professional game-ready inventory icons directly from your Unity assets.

Clockwork Pixie Icon Forge is a Unity Editor tool that transforms prefabs, FBX models, and scene objects into clean PNG icons with automatic framing, lighting presets, batch processing, reusable profiles, and contact sheet generation.

---

## Features
### Automatic Icon Generation
* Generate PNG icons directly from Unity assets
* Transparent background support
* Multiple export resolutions
* Consistent framing across icon sets

### Camera Presets
* Isometric
* Front
* Side
* Top

### Lighting Profiles
* Neutral
* Fantasy
* Sci-Fi
* Stylized
* Minimal

### Item Type Presets
* Generic
* Weapon
* Head Focus
* Shield
* Potion
* Chest
* Resource

### Advanced Composition Controls
* Manual rotation controls
* Manual composition offsets
* Adjustable icon fill
* Preset reset options

### Batch Processing
* Batch export multiple assets
* Add individual assets
* Add project folders
* Automatic contact sheet generation

### Profiles
* Save generation settings as reusable profiles
* Load profiles for consistent results across projects

### Contact Sheets
* Automatically generate overview sheets for exported icon sets
* Quickly review large batches of generated icons

### Debug Tools
* Create preview debug rigs
* Inspect framing and lighting directly in-scene

---

## Quick Start
1. Open **Tools → Clockwork Pixie → Icon Forge**
2. Select a source object
3. Choose an Item Type preset
4. Select a Camera Preset
5. Select a Lighting Profile
6. Adjust composition if desired
7. Click **Export PNG**
